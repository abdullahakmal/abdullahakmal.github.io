Research

IEEE Publication

📰 IEEE · 29 January 2018
Table Recognition in Heterogeneous Documents Using Machine Learning
Published in IEEE Xplore · DOI: 10.1109/document.8270063
Tables are an efficient way to represent structured information. This work proposes a novel learning-based methodology for recognising table contents in heterogeneous document images — particularly challenging where no physical rulings are present. Textual contents are classified as table or non-table elements using a pre-trained neural network with contextual post-processing to correct classification errors. Trained on UNLV and UW3 document image subsets, the system achieved 97%+ accuracy in table element detection.
